# Terminal Cheat Sheet for Chris

## NAVIGATION
- cd ~ (go home)
- cd .. (go up one level)
- pwd (where am I?)
- ls (list files)
- ls -l (list with details)

## FILE OPERATIONS
- cp oldfile newfile (copy)
- mv oldfile newfile (move/rename)
- rm file (delete)
- cat file (view file)

## GIT COMMANDS
- git status (see changes)
- git add . (add all)
- git commit -m "message" (save)
- git push origin main (upload)
- git pull origin main (download)

## DOS TO TERMINAL
dir = ls
cd = cd
copy = cp
move = mv
del = rm
type = cat
cls = clear

## SHORTCUTS
- Ctrl+C = cancel
- Up Arrow = previous command
- Tab = autocomplete
- EOF = exit heredoc

Remember: Real programmers Google constantly!
EOF
